Opportunités par Métier

🎮 Game Designer

OUVERT

Status: Postes actifs en recrutement

OPT Studio recherche des game designers passionnés par la narration interactive et capable de concevoir des mécaniques engageantes pour un public adulte.

Attentes: Portfolio solide démontrant de l'expérience en design narratif, compréhension des jeux interactifs, pensée créative.

👨‍💻 Programmeur

NON MENTIONNÉ

Status: Pas de recrutement signalé

Aucun poste de programmeur n'a été mentionné dans les offres actuelles d'OPT Studio.

Note: Il est possible que le studio utilise des solutions existantes ou dispose déjà d'équipe technique interne.

🎨 Artiste 3D

NON REQUIS

Status: Pas de besoin identifié

OPT Studio se concentre exclusivement sur l'art 2D. Les postes d'artistes 3D ne correspondent pas à la stratégie créative du studio.

Alternative: Proposez vos compétences en 2D pour un meilleur positionnement.

🖼️ UI Artist

PROBABLE

Status: Inclus dans les artistes graphiques 2D

Les artistes graphiques 2D peuvent inclure les UI artists. Excellente opportunité pour un professionnel spécialisé en design d'interface.

Attentes: Design d'interface intuitif, cohérence visuelle, expérience utilisateur.

✏️ Concept Artist

OUVERT

Status: Recrutement actif

Rôle clé dans la création de l'univers visuel d'OPT Studio. Les concept artists jouent un rôle essentiel dans la définition de l'esthétique des jeux.

Attentes: Portfolio riche en concepts originaux, maîtrise du style pour adultes, capacités de création de mondes.
