👨‍💼 Guide par Métier

🎮 Game Designer

Description: Concepteur de gameplay, mécaniques de jeu et expérience utilisateur.

Compétences Clés:

Game Design Document Level Design Gameplay Balancing Storytelling

Formation Recommandée:

  • Licence Jeux Vidéo
  • Master Game Design
  • Portfolio avec prototypes

Conseils Ubisoft:

  • Portfolio avec GDD détaillés
  • Expérience sur moteurs (Unreal, Unity)
  • Passionné par les jeux Ubisoft

💻 Programmeur

Description: Développeur backend/frontend, systèmes de jeu et architecture.

Compétences Clés:

C++ C# Engine Algorithmes Networking

Formation Recommandée:

  • Licence Informatique
  • Spécialisation Game Engine
  • Portfolio GitHub avec projets

Conseils Ubisoft:

  • CV technique très détaillé
  • GitHub/GitLab actif
  • Projets personnels gaming

🎨 Artiste 3D

Description: Modélisateur 3D, texture artist, animateur ou VFX artist.

Compétences Clés:

3DS Max Blender Substance Maya ZBrush

Formation Recommandée:

  • Licence Arts 3D/Animation
  • Spécialisation Game Art
  • Portfolio vidéo showreel

Conseils Ubisoft:

  • Portfolio visuel très pertinent
  • Showreel professionnel HD
  • Assets game-ready optimisés

🖼️ UI/UX Artist

Description: Designer interface, menus, HUD et expérience utilisateur.

Compétences Clés:

Figma Adobe XD Photoshop UX Design Animation UI

Formation Recommandée:

  • Formation UX/UI Design
  • Spécialisation Game UI

Conseils Ubisoft:

  • Cas d'usage game concrets
  • Connaissance ergonomie gaming
  • Prototypes interactifs
