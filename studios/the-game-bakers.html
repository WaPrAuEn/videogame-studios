🎨 Culture du Studio

The Game Bakers est un studio créatif fondé par d'anciens salariés d'Ubisoft, Emeric Thoa et Audrey Leprince. Le studio se spécialise dans le développement de jeux vidéo originaux dans des univers captivants, mêlant créativité artistique et gameplay innovant.

✅ Verdict

The Game Bakers est un studio de développement et d'édition basé à Montpellier, fondé en 2010 par d'anciens talenteux d'Ubisoft. Le studio s'est établi comme créateur de jeux originaux et captivants, avec un portefeuille diversifié incluant Haven, Furi, Squids, Combo Crew et Cairn.

Connu pour son approche créative et artistique, The Game Bakers représente une excellente opportunité pour les professionnels souhaitant rejoindre un studio français dynamique avec une expérience reconnue dans l'industrie. Pour des informations précises sur les postes actuellement ouverts, nous recommandons de consulter directement le site officiel ou de contacter le studio.
