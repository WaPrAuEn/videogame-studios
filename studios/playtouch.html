Détails par Métier

🎲 Game Designer

Niveau de demande : Moyen

Focus : HTML5 et accessibilité

Opportunités : Vous aurez l'occasion de concevoir des jeux accessibles pour un large public, en tirant parti de la plateforme web. Le studio valorise la créativité dans la conception de mécaniques de jeu innovantes et engageantes.

Compétences recherchées : Maîtrise du design HTML5, compréhension de l'accessibilité web, expérience en game design cross-platform.

💻 Programmeur

Niveau de demande : Élevé

Focus : Développement 100% interne en HTML5

Opportunités : C'est le poste le plus demandé. Playtouch privilégie une approche 100% interne pour le développement HTML5, offrant des opportunités d'impact direct sur les productions du studio.

Compétences recherchées : Excellente maîtrise du JavaScript, HTML5 Canvas/WebGL, frameworks modernes (Pixi.js, Babylon.js, Three.js, etc.), optimisation de performance web.

🎨 Artiste 3D

Niveau de demande : Faible

Focus : Spécialisation HTML5, peu de 3D

Opportunités : Peu de postes disponibles. Le studio privilégie les technologies web légères plutôt que la 3D complète. Si vous avez une spécialisation en HTML5 et graphismes web, vous pourriez trouver des opportunités.

Compétences recherchées : WebGL, Three.js, Babylon.js, optimisation d'assets pour le web, expérience avec les jeux HTML5.

🖼️ UI Artist

Niveau de demande : Élevé

Focus : UI pour jeux web cross-platform

Opportunités : Forte demande. Vous concevrez des interfaces utilisateur pour des jeux web accessibles sur tous les appareils. L'expérience en responsive design est un plus.

Compétences recherchées : Design UI/UX, responsive design, connaissance des limitations des navigateurs, outils de design modernes (Figma, Adobe XD), maîtrise du CSS/HTML5.

✨ Concept Artist

Niveau de demande : Moyen

Focus : Concepts pour jeux fun et innovants

Opportunités : Le studio a besoin de créateurs de concepts pour imaginer des jeux fun et innovants. Vous collaborerez étroitement avec les game designers.

Compétences recherchées : Illustration digitale, concept art pour jeux, maîtrise de logiciels comme Photoshop/Procreate, capacité à traduire les idées en visuels engageants.

⭐ Verdict & Conseils de Candidature

Profil Idéal

  • Compétences solides en HTML5 et JavaScript
  • Portfolio de jeux web ou projets web démontrant votre expertise
  • Motivation pour travailler dans un environnement international (France/Maurice)
  • Passion pour les jeux accessibles et cross-platform
  • Capacité à s'adapter à une petite équipe dynamique (1-50 employés)

Difficulté de Candidature

Niveau : Moyen

Playtouch est une petite équipe internationale qui privilégie l'expérience HTML5. Les postes de Programmeur et UI Artist offrent les meilleures opportunités. L'expérience en jeux web et la démonstration d'une compréhension des enjeux d'accessibilité seront des atouts majeurs.
