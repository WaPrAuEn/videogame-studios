Opportunités par Métier

🎮 Game Designer

Niveau : ÉLEVÉ

Description :

Studio axé sur des jeux narratifs avec action et plateforme. Les designers travaillent sur des franchises AAA avec une forte composante narrative.

Compétences requises :

  • Maîtrise du game design narratif
  • Expérience sur projets AAA
  • Expertise gameplay action/plateforme
  • Portfolio solide avec jeux publiés

💻 Programmeur

Niveau : ÉLEVÉ

Description :

Développement de franchises AAA complexes sur PlayStation. Architectures logicielles sophistiquées et optimisation de performances.

Compétences requises :

  • C++ et architecture engine avancée
  • Optimisation pour PlayStation
  • Expérience projets AAA complexes
  • Maîtrise des systèmes de gameplay avancés

🎨 Artiste 3D

Niveau : ÉLEVÉ

Description :

Création d'assets 3D pour jeux riches en graphismes comme Spider-Man et Ratchet & Clank. Standards visuels très élevés.

Compétences requises :

  • Maya, Blender ou 3DS Max
  • Modélisation haute/basse poly
  • Texturing AAA haut de gamme
  • Connaissance pipeline temps réel

🖼️ UI Artist

Niveau : MOYEN À ÉLEVÉ

Description :

Création d'interfaces pour jeux PlayStation. Design intuitif et esthétique alignée avec l'identité des franchises.

Compétences requises :

  • UI/UX design solide
  • Adobe Creative Suite
  • Expérience jeux console
  • Animation UI et motion design

✏️ Concept Artist

Niveau : ÉLEVÉ

Description :

Création et définition visuelle de franchises iconiques. Concept art pour personnages, environnements et univers.

Compétences requises :

  • Dessin traditionnel et digital expert
  • Photoshop, Painter ou Procreate
  • Direction artistique AAA
  • Portfolio de concept art impressionnant
